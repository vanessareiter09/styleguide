Styleguide TiMe

Introduction

Hello and welcome to our styleguide for the mmp2a. We hope you like it :)

TiMe is a plattform where you can easily track your working hours and those hours you work for a client or project. TiMe combines all the advantages of a simple time tracking system and a complex ticket management system. As TiMe is so extensive, it provides the perfect plattform to work easy, efficient and structured.

The styleguide is made with Jekyll and the code is based on the source of Jerome Coupe. Further information about the styleguide as well as a explanation on how to run the project can be found on GitHub.
