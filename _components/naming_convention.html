Naming Convention

  • our class names are written in lowercase and separated by dashes
    • .welcome-message { text-align: center; }
  • we also took care that the scss files have meaningful names
    for example the welcome page has it's on scss file named welcome.scss
  • we tried to stick to BEM notation
    BEM has only class selectors, but we have for example also an id for the active link
  • we avoided specifying units for zero values
    • body { margin: 0; padding: 0; }
  • for colors we also used hex format and variables. Note: in ruby on rails we have to declare varibales in a css file
    • :root { --primary: #58797b; }
    nav { background-color: var(--header); }
